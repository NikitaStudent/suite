Техническое задание :Векторное произведение векторов

Информация о проекте

Проект предназначен для нахождение векторного произведения онлайн.

Предусмотрены следующие возможности:

  • Ввод данных пользователем
  • Отображение результата

Описание

Проект представляет собой Web-приложение, содержащее страницу с формами для ввода данных. После ввода данных и нажатия кнопки управление передается PHP-скрипту, который по результатам валидации данных сгенерирует страницу-результат.

Стартовая форма содержит следующие элементы :

  • Форма для ввода значений вектора.
  • Кнопка для запуска PHP-скрипта

В случае положительного результата валидации скрипт начнет отображение значений для выбранной операции и сгенерирует новую страницу с ее графическим представлением

Технологии

  • Для разметки форм и отображения данных на стороне клиента :
    • HTML
    • CSS
  • Серверная часть :
    • PHP
